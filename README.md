## 클레이 스테이킹 웹 서비스

### 요구사항
```
  - 회원가입 시 자동 지갑 생성 및 생성 직후 0.1 Klay 지급
  - 회원 간 / 외부 지갑에 클레이, 토큰 전송 기능
  - 사용한 API, 트랜잭션 확인
  - 스테이킹 상품 제공 (30일 동안 00:00시마다 일정 수량 Klay 지급)
```


### 개발 & 테스트 & 배포환경
```
  - Spring Boot
  - Spring Security + JWT
  - Spring Batch
  - Data JPA
  - Thymeleaf
  - H2 (Local)
  - MariaDB (Production)
  - Redis
  - AWS
  - Docker
  - Swagger
  - Klaytn API
  - Junit5
  - Mockito
```
